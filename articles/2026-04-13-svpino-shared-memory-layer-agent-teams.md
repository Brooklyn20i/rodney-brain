## Summary
Santiago amplifies the "agent amnesia" problem: when a Claude session ends, everything important disappears — prompts, tool calls, decisions, traces. Points to a shared memory layer that spans sessions and is available to the entire team, capturing all agent actions and making them searchable. "We need infrastructure like this across the board now."

## Why It Matters
Three separate voices (this, Akshay's agent memory article, Harrison Chase's harness/memory piece) are converging on the same point: memory infrastructure is the missing layer in agent deployment. Agent amnesia is an emerging enterprise risk, not a theoretical concern. The architectural question is centralised shared memory vs per-agent isolated memory — this article argues strongly for shared.
