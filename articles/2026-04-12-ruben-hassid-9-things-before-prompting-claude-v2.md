Don't type another prompt into Claude. Do these 9 simple things first:

1. Download the Claude app — claude.com/download. The desktop app is better than the web.

2. Pick the Right Model — Select Opus 4.6. Turn on Extended Thinking. Click models → Opus 4.6 → Extended Thinking ON.

3. Use Cowork, Not Chat — Cowork reads your actual files. Chat doesn't. Point Claude to your project folder. It reads everything before responding.

4. Build Your Folder Structure — Create 4 folders: ABOUT ME, PROJECTS, TEMPLATES, CLAUDE OUTPUTS.

5. Write Your "About Me" md File — .md file about who you are, your work, thinking. Claude reads it before every response. Less prompting, better output.

6. Write Your Anti-AI Style File — A .md file listing every word & pattern you refuse. Ban "delve," "landscape," "leverage." Claude follows it. You stop sounding like a robot.

7. Start Asking — End prompt with: "Ask me clarifying questions first." Claude generates a form. You answer. The output jumps 2–3 levels.

8. Set Up Global Instructions — Go to Settings → Profile → Global Instructions. Write how you write, what you hate & like. Claude reads this before every single conversation.

9. Edit Your Prompt — Don't send a new message. Edit the first one. Every follow-up eats tokens. Editing keeps your context clean.
